## Installation

1. Create Virtual Environment:
    ```
    python -m venv env 
    ```

2. Activate Virtual Environment:
    ```
    .\env\Scripts\activate
    ```
2. Install the required Python libraries:
    ```
    pip install -r requirements.txt
    ```
